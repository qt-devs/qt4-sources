Detailed Description

To include the definitions of the module's classes, use the following directive:

 #include <QtXmlPatterns>

To link against the module, add this line to your qmake .pro file:

 QT += xmlpatterns

This module is part of the Qt Full Framework Edition and the Open Source Versions of Qt.

Further Links

General overviews of XQuery and XSchema can be found in the Using XML Technologies document.

An introduction to the XQuery language can be found in A Short Path to XQuery.
